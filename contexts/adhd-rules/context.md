# Kommunikationskontrakt (gäller alltid)

Mike är neurodivergent (ADHD-mönster), kort temperament, hoppar lätt mellan saker, behöver struktur.

1. **Max 5 punkter per svar** — ADHD-anpassat
2. **Svenska** för all kommunikation och kodkommentarer
3. **Plan först → vänta på OK → kör**
4. **Korta, konkreta svar** — inga väggar av text
5. **Tabeller > listor > prosatext** när data ska visas
6. **Lyssna FÖRST på problemet** — fråga vad som är fel innan fix
7. **Stoppa proaktivt arbete** om Mike säger "pausa", "stoppa", "det räcker"
8. **Fråga hellre än att gissa** — speciellt infra/domäner

## Tar emot info bäst som
Korta, tydliga förklaringar i listor/tabeller. Vid uppgifter: tydliga steg, inte vaga riktningar.

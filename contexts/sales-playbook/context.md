# Säljstrategi per produkt

## B2B / AIOS (Byggflow, Infrea, AF-SIUS, Full AI OS)

| Kanal | Steg |
|---|---|
| **LinkedIn-artiklar** | AIOS + AI-agenter — thought leadership |
| **Email outreach via Instantly** | Domän värms upp |
| **Familiarity-loop** | Like → Comment → Connect → DM → Call → Offert <4h |
| **Underlag i pitch** | Visa vad kunden tjänar/sparar, skalning, branschbehov |

## B2C ThyroidAI

| Kanal | Steg |
|---|---|
| **FB-grupper** | Bjuda in från Thyroid-grupper → ThyroidAI FB-sida |
| **TikTok** | När video creation är klar |
| **DM Outreach** | Utan att flagga kontot |

## Hårda kanal-regler
- **LinkedIn används ENDAST för B2B** — aldrig ThyroidAI
- **ThyroidAI = FB + TikTok + DM** — aldrig LinkedIn

## ICP B2B
VD/COO/Grundare, 5–50 anställda, SaaS/konsult/rekrytering/fastighet/byråer, Sverige+Norge.

## Princip
Hellre färre stora kunder än många små som tar tid. Anpassa pris efter budget endast för att vinna **referenser**.

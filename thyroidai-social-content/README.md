# ThyroidAI — Social Content System

> **Syfte:** Allt du behöver för att producera igenkännbart, viralt short-form content för ThyroidAI på löpande band — samma karaktärer, samma stil, varje video.
> **Språk i content:** Engelska (internationell räckvidd). **Format:** 9:16, <60s.
> **Skapad:** 2026-06-17

---

## Den röda tråden (i en mening)

> **Dr. Vee — en torr, vass läkare — roastar den trasiga sköldkörtelvården som miljontals (mest kvinnor) känner igen, med fakta i ryggen. Picture-in-picture. Samma karaktär varje gång. Bara stilen (realistisk ↔ animerad) flippar mellan videos.**

---

## Snabbstart (när du vill spela in)

1. Läs `01_CHARACTERS.md` → generera Dr. Vee (en gång, spara seed/referensbild).
2. Öppna `scripts/BATCH-01.md` → välj manus #01.
3. Följ `02_VIDEO_SPEC.md` för PiP + captions + flip-typ.
4. Rendera enligt `03_PRODUCTION_PIPELINE.md`.
5. Posta enligt kanaler i `00_CONTENT_BIBLE.md` (FB-grupper + TikTok — **aldrig LinkedIn** för ThyroidAI).

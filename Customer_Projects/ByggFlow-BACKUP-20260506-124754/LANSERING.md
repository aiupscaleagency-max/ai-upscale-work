## 🌐 KOPPLA DOMÄN (2 min – gör nu)

### Alternativ A: demo.aiupscale.agency (rekommenderas)
Lämnar root-domän fri för din landningssida.

1. Gå till https://vercel.com/aiupscaleagency-2989s-projects/byggflow-demo/settings/domains
2. Skriv `demo.aiupscale.agency` i fältet → klicka **Add**
3. Vercel visar 1 DNS-record att lägga till hos din domän-registrar:
   ```
   Typ:   CNAME
   Namn:  demo
   Värde: cname.vercel-dns.com
   ```
4. Logga in hos registrar (Loopia/One.com/etc), lägg in record, vänta ~5 min
5. Ditt demo finns nu på **https://demo.aiupscale.agency** ✓

### Alternativ B: aiupscale.agency (root)
Om du vill ha root-domänen för demon (bygg landningssida på subdomän senare):

1. Samma sida → skriv `aiupscale.agency`
2. Vercel ger 2 records (en A-record + en TXT för verifiering)
3. Lägg in båda hos registrar

**Tips:** Du kan ha BÅDA aktiva — `byggflow-demo.vercel.app` fungerar alltid som backup.

## 🎬 DEMO-VIDEOSKRIPT (3 min — kör i denna ordning)

### Inledning (15 sek)
> "Det här är ByggFlow — ett system jag byggt för byggbranschen. Hantverksfirmor får live-koll på fältarbete, kommuner får full insyn, och ingen luras på en enda krona. Låt mig visa."

### Akt 1 — Hantverkar-vy mobil (30 sek)
1. Öppna `/falt` på mobil eller mobil-läge
2. Visa: ruttkarta med 6 stopp + GPS-puls
3. Klicka pågående jobb (00:32 timer som tickar)
4. **"Hantverkaren checkar in med ETT klick. GPS skickas till admin direkt."**
5. Klicka "Dagbok" → fyll snabbt → spara → toast "syns nu i Kommunportalen"

### Akt 2 — Admin Dashboard (40 sek)
1. Klicka "Admin" i navbar
2. **Peka på live-panelen:** "Total portfölj 279 miljoner. Bruttomarginal 42 miljoner. Detta uppdateras varje gång en hantverkare loggar tid."
3. Klicka "Live i fält" — högerspalten — välj Anders Lindqvist → modal öppnas
4. Visa **timkostnad-blocket** överst: "Inköp 385 kr/tim, debiterat 695 kr — 45% marginal. Live."
5. Klicka **ÄTA-fliken**: "18 ÄTA-poster, värde +5,3 M kr. Två väntar beslut."

### Akt 3 — Koncernstruktur (20 sek)
1. Sidomeny → "Koncernstruktur"
2. **"Moderbolag 487 miljoner i omsättning, 7 dotterbolag."**
3. Klicka Karlsson Bygg → expandera avdelningar
4. Klicka "Lönsamhet" → "Per anställd" — **"Du ser exakt vem som drar in mest marginal."**
5. Klicka "HR & Certifikat" — **"4 certifikat går ut inom 30 dagar — du missar aldrig ett ID06."**

### Akt 4 — Kommunportalen (50 sek) **← VIKTIGAST**
1. Navbar → "Kommunportal" → "Logga in som Stockholms Kommun"
2. **"Beställaren ser allt i realtid — inga månadsrapporter, ingen mailpingpong."**
3. Klicka projektkort "Stamrenovering Kv. Björken" → modal öppnas
4. Visa **AMA-fliken**: "Beställaren kan räkna eget anbud — ändra mängd eller pris, diff syns live."
5. Sidomeny → "Anbudsförfrågningar" → klicka UPH-2026-014
6. **"Fyra anbud. Lägsta priset markerat. Ett klick för att tilldela."**
7. Sidomeny → "Granska fakturor" → visa kontrollchecklistan
8. **"Faktura returnerades för att timmarna inte stämde mot dagboken — systemet upptäckte det automatiskt."**
9. Sidomeny → "Leverantörsbetyg" → **"Bravida AB: avråder. Det här syns nästa gång ni upphandlar."**

### Avslut (15 sek)
> "Hela demon är skarpt fungerande, byggd på 2 dagar med AI. Vi skräddarsyr för varje kund — er logga, era projekt, era anställda. Boka 15 min så visar jag exakt hur det skulle se ut för er."

| Rapporter | Slutbesiktningar, avvikelser, veckorapporter |
| Fakturor | Länk till skapa ny + status-summering |
| Material | Lager + förbrukning, "Beställ"-knapp på låga artiklar |
| Anställda | **Inköp/tim · Debiterat/tim · Marginal kr+%** per person |
| Ruttplan | Karta + alla aktiva positioner |
| **Koncernstruktur** | Moderbolag 487M → 7 dotterbolag → avdelningar |
| **Lönsamhet** | Ranking per projekt/kund/anställd |
| **HR & Certifikat** | ID06, heta arbeten med utgångsvarning |

### Kommunportal (beställare)
| Vy | Innehåll |
|---|---|
| Översikt | Live-panel + sammanfattning + karta + Vem jobbar nu |
| Projekt | Tabell med kommunens 12 projekt |
| Dagbok | Hantverkarnas dagliga inlägg från fältet |
| Rapporter | Slutbesiktningar, avstämningar |
| Fakturor | Översikt över alla fakturor till kommunen |
| Material | Lager + förbrukning |

| Karta | Live-positioner |

| **Leverantörsbetyg** | 8 firmor, 5 kriterier, Rekommenderas/Avråder |

### Hantverkar-vy (mobil)
- Ruttplan med 6 stopp + GPS-puls
- Live-timer på pågående jobb
- "Navigera"-knapp → öppnar Google/Apple Maps
- Vägbeskrivning till nästa stopp (svensk turn-by-turn)
- Sticky Dagbok + Material-knappar

### Faktura-vy
- 10 fakturor med status-badges
- Skapa ny med ROT/RUT-toggle, automatisk beräkning
- PDF-förhandsvisning med ByggFlow-design

### Per projekt — full transparens (klick på vilket projekt som helst)
1. **Projekt-live-panel** överst (budget · fakturerat · kvar · marginal)

5. **AMA & Mängder** (räkna eget anbud live)

7. **Material** (förbrukat)
8. **Rapporter** (status)
